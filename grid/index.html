Bootstrap Grid!

Using col-1 to col-12

In Bootstrap each row can have 12 columns, but how is it?
It means that in Bootstrap the row takes the 100% of the container, and each column has a width of 8.333333% (because 100% / 12 = 8.333333%)
So, if we put a column with the class col-1 the max-width will be 8.333333% (because 8.333333% * 1 = 8.333333%)
If we put a column with the class col-2 the max-width will be 16.666667% (because 8.333333% * 2 = 16.666667%)
And so on and so forth
